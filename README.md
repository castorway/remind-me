# RemindMe

A website for easily setting and accessing reminders, with authentication and SMS notifications.

Made for an AlbertaSat Ground-Station Software Kickoff Project.

## Dependencies:
* flask
* flask-sqlalchemy
* flask-login
* flask-apscheduler
* twilio
