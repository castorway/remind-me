# RemindMe
A website for setting regular reminders, with authentication and SMS notifications.

Made for an AlbertaSat Ground-Station Software Kickoff Project.

## Dependencies:
* flask
* flask-sqlalchemy
* flask-login
* flask-apscheduler
* twilio
